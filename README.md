Análisis de datos aeronáuticos con Python y Power BI (limpieza, transformación y visualización)
# ✈️ Análisis de Datos Aeronáuticos

Este proyecto forma parte de mi formación como Data Analyst, enfocado en el análisis de datos del sector aeronáutico.

## 📊 Objetivo

Realizar limpieza, transformación y análisis de datos para identificar patrones relevantes y generar insights.

## 🛠️ Herramientas utilizadas

- Python (Pandas)
- Power BI
- Jupyter Notebook

## 🔍 Procesos realizados

- Limpieza de datos (valores nulos, inconsistencias)
- Normalización de variables
- Análisis exploratorio (EDA)
- Transformación de datos
- Visualización de información

## 📈 Resultados

Se lograron identificar patrones en los datos que permiten comprender mejor la estructura y comportamiento del dataset.

## 🚀 Próximos pasos

- Integración con SQL
- Automatización de procesos ETL
- Mejora de visualizaciones
