# ✈️ Análisis de Datos Aeronáuticos

Proyecto de análisis de datos enfocado en el estudio de aeropuertos y tráfico de pasajeros, utilizando Python y Power BI.

---

## 🎯 Objetivo

Analizar datos aeronáuticos para identificar patrones en el tráfico aéreo, distribución geográfica de aeropuertos y posibles insights operativos.

---

## 🛠️ Herramientas utilizadas

- Python (Pandas, Numpy)
- Jupyter Notebook
- Power BI

---

## 🔍 Proceso de análisis

1. Carga y exploración inicial de datos  
2. Limpieza de datos (valores nulos, formatos inconsistentes)  
3. Transformación de variables  
4. Análisis exploratorio (EDA)  
5. Visualización de datos  

---

## 📊 Insights principales

- Existe una concentración de tráfico en determinados aeropuertos clave  
- Se identifican diferencias relevantes entre tipos de aeropuertos  
- Los datos presentan inconsistencias iniciales que requieren limpieza previa  

---

## 📁 Archivos del proyecto

- `analisis_aeropuertos.ipynb` → análisis completo en Python  
- `dashboard_powerbi.png` → visualización en Power BI  

---

## 📌 Conclusión

El análisis permite comprender la estructura del sistema aeroportuario y demuestra la importancia de la limpieza y transformación de datos para obtener información útil.
